# Project Images

Drop your real project photos here, one folder per project.

## Folder naming
Use a descriptive slug that matches the project, e.g.:
  project-1/
    main.jpg          ← used as the thumbnail
    before.jpg
    after.jpg
    detail.jpg

## How to reference in the admin
Once you drop images in (e.g. project-1/main.jpg), set the URL in the admin panel to:
  /images/projects/project-1/main.jpg

Vite serves everything inside public/ at the root path during dev and in production builds.

## Recommended image sizes
- Thumbnail: 800 × 500 px (16:10 ratio)
- Carousel images: 1400 × 800 px (16:9 ratio, min 1MB for good quality)
